문제 1.
```css
public int CompareSolution(int num1, int num2)
{
    return num1 == num2 ? 1 : -1; 
}
```
반환값이 1 또는 -1 둘로 나타나기에 조건연산자인 ? : 사용한다.

```
public int Divsolution(int num1, int num2)
{
    float div = (float)num1 / num2;

    return (int)(div * 1000);
}
```
